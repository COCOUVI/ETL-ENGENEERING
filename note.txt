En résumé : Le client S3 sert de pont entre ton code Python et MinIO pour que tu puisses utiliser MinIO exactement comme si c’était AWS S3, avec authentification, requêtes simplifiées et gestion d’erreurs.


boto3 est la bibliothèque Python officielle pour parler à S3.

On utilise boto3 pour permettre à une application Python de communiquer avec un stockage objet compatible S3 (comme MinIO ou AWS S3) afin d’uploader, télécharger et gérer des fichiers de façon scalabl



get_S3_client
Cette fonction crée un client S3 avec boto3 configuré pour pointer vers MinIO afin de permettre à mon application Python d’uploader et de gérer des fichiers dans le stockage objet.
